# Recursion Practice in Python

This repository contains Python programs demonstrating **recursion** through basic examples.  
All functions are implemented in a single file and executed with sample inputs.

---

## Implemented Programs

### 1. Factorial Using Recursion
- Calculates factorial of a number
- Handles base cases `0` and `1`

**Example Calls:**
- `factorial(7)`
- `factorial(0)`
- `factorial(5)`

---

### 2. Sum of List Elements Using Recursion
- Returns sum of all elements in a list
- Returns `0` for an empty list

**Example Calls:**
- `sum([1,2,3,4,5])`
- `sum([])`
- `sum([10,20,30])`

---

### 3. Fibonacci Series Using Recursion
- Computes the nth Fibonacci number

**Example Call:**
- `fibonacci(7)`

---

### 4. Reverse a String Using Recursion
- Reverses a string
- Handles empty string

**Example Calls:**
- `rev_str("string")`
- `rev_str("")`

---

### 5. Power of a Number Using Recursion
- Calculates `x` raised to power `n`

**Example Call:**
- `pow_num(3, 7)`
---

## Purpose
This repository is created for **practice and understanding of recursion concepts** in Python.
